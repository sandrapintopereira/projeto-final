# Frame Marcado — Watchlist de Filmes & Séries

[![CI](https://github.com/sandrapintopereira/projeto-final/actions/workflows/ci.yml/badge.svg)](https://github.com/sandrapintopereira/projeto-final/actions/workflows/ci.yml)

Projeto final desenvolvido para o módulo de Programação em JavaScript.

## Descrição

Aplicação SPA (Single Page Application) desenvolvida em Angular para gerir uma watchlist pessoal de filmes e séries. Nunca mais fiques sem saber o que já viste ou o que tens para ver.

## Funcionalidades

- Dashboard com indicadores em tempo real (melhores avaliações, filmes e séries vistos)
- Lista completa com pesquisa, filtros e ordenação
- Página de detalhe por filme/série
- Adicionar e editar conteúdos com validação de formulário
- Estado visual — visto vs por ver

## Tecnologias

- Angular 20+
- TypeScript
- HTML & CSS

## Instalação
No terminal: 
- npm install -g @angular/cli
- npm install
- ng serve

Abre o browser em `http://localhost:4200`
